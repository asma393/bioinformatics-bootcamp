# Bioinformatics Bootcamp
This repository contains my training exercises in Linux and bioinformatics.
## Project 1: Cancer Mutation Analysis
Tools used:
- grep
- cut
- sort
- uniq
- awk
  
Dataset:
Simulated cancer mutation dataset including genes such as TP53, BRCA1, EGFR and KRAS.
Analysis performed:
- extraction of gene names
- mutation counting
- mutation filtering by gene
- generation of summary files
